## What Front Matter Should Include

According to Material for MkDocs best practices, page front matter should include:

```yaml
---
title: Page Title
description: Brief description for SEO and search
---
```

Optional fields:

- `icon: material/icon-name` - Icon for section index pages
- `search.boost: 2` - Boost search ranking for important pages
- `hide: [navigation, toc]` - Hide navigation or TOC
- `tags: [tag1, tag2]` - Tags for grouping

## Impact

**Current State:**

- No metadata for SEO (search engines, social sharing)
- No search boosting for important pages
- No icons for section pages
- Missing structured metadata

**Material Theme Expectations:**

- Section index pages (docs/\*/index.md) should have icons
- Important pages (quickstart, key concepts) should have search boost
- All pages benefit from explicit title/description
